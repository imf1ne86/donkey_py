# donkey_py

Перенос игры Donkey (QBasic) на Python 3 с использованием библиотеки Pygame. Код сохраняет игровую логику (две полосы, автомобиль, препятствия - ослики), крупную "пиксельную" графику с масштабом, простые звуки и управление: пробел - смена полосы, Esc - выход. Скопируйте в файл donkey.py, установите Pygame (pip3 install pygame) и запустите: python3 donkey.py

Примечание: звук реализован простыми тонами (генерация синусоидальных сэмплов). Графика выполнена через масштабированные спрайты для эффекта "больших пикселей".
